이 문서는 **SOOL**과 **스키장 혼잡도 분석 프로젝트**의  
기술적 설계와 데이터 구조를 정리한 기술 문서입니다.

## 목적
- 구현보다 **설계와 구조를 우선**
- 기능 추가 전에 **의사결정 기준을 명확히**
- 블로그·포트폴리오로 재활용 가능한 문서화

## 프로젝트 구성

### SOOL
- 전통주 데이터 기반 분석 및 추천 시스템
- FastAPI + SQLite + Next.js 기반 MVP

### Ski Resort Analysis
- 이미지 기반 혼잡도 분석 프로젝트
- ffmpeg 기반 데이터 수집 및 분석
